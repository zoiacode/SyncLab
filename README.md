# SyncLab

Disponibilizar uma plataforma web que centralize a gestão de salas, 
equipamentos e eventos, com agendamento, calendário integrado e 
notificações úteis por perfil.  
A gestão de salas, laboratórios, equipamentos e eventos muitas vezes é 
descentralizada, gerando dificuldades de comunicação e organização. Público: 
alunos, professores e pessoal administrativo
